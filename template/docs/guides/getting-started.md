# Getting Started

Add Markdown files under `docs/` and they appear automatically in the sidebar and MCP tools.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000 to browse docs in the browser, or connect an MCP client to `/mcp`.

## Writing pages

- Use folders to group related pages
- Name files with kebab-case (`error-handling.md`)
- Add an `index.md` at the root for homepage content
